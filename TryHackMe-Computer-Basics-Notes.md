[Task 2] INSIDE A COMPUTER SYSTEM

1. Hardware Components:
   - CPU: Processing unit, computer ka dimagh
   - RAM: Temporary memory, running programs ke liye
   - Storage: Hard drive/SSD, data save karne ke liye
   - Motherboard: Sab parts ko connect karta hai

2. Software Components:
   - Operating System: Windows, MacOS, Linux
   - Application Software: MS Office, Browser, etc
   - Hardware aur Software mil kar system banate hain

[Task 3] WHAT HAPPENS WHEN YOU PRESS THE START BUTTON?

1. Power button dabane se motherboard active hota hai
2. BIOS/UEFI start hota hai aur hardware check karta hai
3. BIOS hard drive se Operating System load karta hai
4. OS load hone ke baad Desktop screen aati hai
5. Is process ko "Booting" kehte hain
6. OS kharab ho to computer on nahi hota
